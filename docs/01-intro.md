# Bài 1: Giới thiệu React 

## React là gì ? 

React là một thư viện javaScript mã nguồn mở, do Meta phát triển, 
dùng để xây dựng giao diện người dùng(UI) cho các ứng dụng web hiện đại
Thay vì viết HTML tĩnh, ta chia UI thành các **component** - mỗi component là 1 phần nhỏ có thể tái sử dụng. 

## Lợi ích: 
 - Dễ mở rộng, dễ bảo trì. 
 - Tái sử dụng code (component). 
 - Tối ưu hiện suất với Virtual DOM.
